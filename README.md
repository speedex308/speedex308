# ⚡ Speedex308

**Speedex308** — это минималистичный форум в стиле скорости, созданный для быстрых и удобных обсуждений.

## 🚀 Возможности

- 📝 **Создание постов** с заголовком, текстом и фото
- 💬 **Комментарии** к постам
- ⬆️⬇️ **Система голосования** (лайки/дизлайки)
- 🔞 **NSFW-фильтр** — автоматическая маркировка контента 18+
- 🌐 **Языковой фильтр** — показ постов на русском или английском
- 👤 **Профиль пользователя** с настройкой имени
- 📱 **Адаптивный дизайн** под телефоны и ПК
- 🎨 **Тёмная тема** с анимированными линиями скорости

## 🛠 Технологии

- HTML5, CSS3, JavaScript
- localStorage для хранения данных
- Netlify для хостинга

## 📦 Установка

1. Скачай файл `index.html`
2. Открой в браузере — форум готов к работе

## 🌍 Деплой

Сайт доступен по адресу: [speedex308.netlify.app](https://speedex308.netlify.app)
